# Game Of Life

A game of life clone made for my Honors AI course.

## Installation

```bash
# install dependencies
yarn

# 1. standalone app (verison 1)
yarn run start

# 2. web version (version 2)
yarn run dev
```

If you ran the web version, visit `localhost:3000`.
